## QPulse App (Project Details):
A lightweight, real world scenario agentic workflow demo for collecting quick group sentiment and choices in minutes.

## What it is:
QPulse is a simple “ask → collect → summarize” system that lets a facilitator post a question, collect anonymous answers from a Telegram chat, and generate a results summary (including a winner + insights).
